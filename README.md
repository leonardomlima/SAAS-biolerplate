Boilerplate SaaS com FastAPI + React, multi-tenant, billing via ASAAS e email via Brevo.

## Quickstart
- `make up`
- `make migrate`
- `make seed`

## B2B/B2C
Para B2C, mantenha tenant único e desative TenantSwitch.
